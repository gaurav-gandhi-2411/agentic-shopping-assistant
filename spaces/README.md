# Agentic Shopping Assistant

A multi-turn conversational shopping assistant over 20,000 H&M fashion articles.

**Ask things like:**
- *"show me black summer jackets"*
- *"something light and breathable for the beach"*
- *"compare the first two you showed me"*
- *"anything in blue instead?"*

Built with LangGraph · FAISS + BM25 hybrid retrieval · Groq (Llama 3.1 8B)

> **Source code:** [github.com/gaurav-gandhi-2411/agentic-shopping-assistant](https://github.com/gaurav-gandhi-2411/agentic-shopping-assistant)
